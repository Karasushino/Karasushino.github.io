Projects

UE4 Axe Throw (C++ & BP)

Implemented an Axe Throw mechanic heavily inspired on God of War.

  • My work is the axe behaviour and functionality in C++.
  • Axe calculates the number of spins based on distance from the player.
  • Axe correctly lodges on any surface, including angled surfaces.
  • Easily change curvature, speeds and rotations from the editor.
  • A lot of attention to detail and polish, the main purpose was to take a simple mechanic concept and make it complex.
  • One of the most valuable things I learned during this project besides developing my skills with Unreal Engine is the importance of using debugging tools to find bugs faster, after this project I started creating and using debugging elements that sped up my workflow.

DX11 Shaders (C++ & HLSL)

Coded a dynamic graphical application utilizing DX11 shaders.

  • Trochoidal waves with correct lighting that receive shadows of other geometry and the water color is based on depth. (with dynamic tessellation).
  • Swappable tessellated terrain generation based on heightmaps. (also has dynamic tessellation).
  • Grass shader with wind simulation, random blade rotation, height and spacing per vertex.
  • Pointlights and Directional lights with shadow maps.
  • Most shader parameters can be changed with ImGui in runtime.
  • In this project, I developed my C++ skills a lot and learned about memory management between the CPU and the GPU, in some cases the framework that the university provided wasn't enough so I extended some parts of it.
  • Source code available by clicking the Github image.
